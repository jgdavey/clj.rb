# clj.rb - Tools for using JRuby from Clojure

## Installation

clj.rb is published to [Clojars](https://clojars.org/clj.rb). To use from lein/boot:

    [clj.rb "0.1.0"]

From maven:

    <depencency>
      <groupId>clj.rb</groupId>
      <artifactId>clj.rb</artifactId>
      <version>0.1.0</version>
    <dependency>
